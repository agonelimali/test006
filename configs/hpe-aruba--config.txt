=== NETWORK ARCHITECTURE OVERVIEW ===

Device Models and Roles:
- Core Router: Aruba 8325-32C - Primary routing, VSX, and backbone connectivity
- Distribution Switches: Aruba 8320-48Y8C (x2) - VLAN aggregation, L3 routing, redundancy
- Access Switches: Aruba 6300M-48X (x4) - End-user connectivity, port security, 802.1X enforcement
- Firewall: Aruba 9004 Gateway - Network security, access control, micro-segmentation enforcement

Network Topology Description:
- Three-tier hierarchical design with redundant core and distribution
- Core provides high-speed backbone and VSX-based redundancy
- Distribution aggregates access switches, provides L3 routing, and connects to core via redundant trunks
- Access switches connect end-user devices, enforce 802.1X, and apply NAC policies
- Firewall enforces micro-segmentation, least-privilege, and integrates with SIEM for monitoring and automated response
- All inter-device links are redundant trunks; all management and logging is centralized
